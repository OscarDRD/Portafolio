# Portafolio de Backend Developer

Este es un portafolio creado con [Astro](https://astro.build) y [Tailwind CSS](https://tailwindcss.com), diseñado para mostrar proyectos de Backend.

## Requisitos Previos

Necesitas tener instalado [Node.js](https://nodejs.org/) (versión 18 o superior) para ejecutar este proyecto.

## Instalación y Ejecución

1.  Abre una terminal en la carpeta del proyecto.
2.  Instala las dependencias:

    ```bash
    npm install
    ```

3.  Inicia el servidor de desarrollo:

    ```bash
    npm run dev
    ```

4.  Abre tu navegador en `http://localhost:4321` para ver el portafolio.

## Estructura del Proyecto

-   `src/pages/index.astro`: La página principal del portafolio.
-   `src/layouts/Layout.astro`: El diseño base de las páginas.
-   `src/components/`: Componentes reutilizables.

## Personalización

Puedes editar el archivo `src/pages/index.astro` para agregar tu información personal, habilidades y proyectos.
